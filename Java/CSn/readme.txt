To run programs in this library, navigate to this
directory and type (for example):

java -cp CSn.jar edu.normandale.csn.CompareSorts 64000 3 is
java -cp CSn.jar edu.normandale.csn.CompareSorts 2048000 3 mqh

Or

java edu.normandale.csn.CompareSorts 1024000 3 mqh

To create CSn.jar:
1. If getting errors on JUnit imports:
    a. CLick the Testing icon.
    b. Enable JUnit
2. Open Java Projects
3. Click the arrow icon to build the .jar file
4. Select <without main class> at end of list
